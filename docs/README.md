# Docs Module

This directory is migrated from `spring-ai-docs/src/main`.

## Structure

- `antora/`: Antora component and playbook
- `asciidoc/`: additional AsciiDoc/Markdown content
- `javadoc/`: javadoc overview assets

## Commands

From repository root:

```bash
pnpm install
pnpm docs:build
pnpm docs:clean
```

Generated site output is written to `docs/site`.
